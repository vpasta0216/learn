## 1. 개요

### 1.1 목적
매일 아침 6:30에 카테고리별 인기 뉴스를 자동으로 큐레이션하여 제공하는 웹 앱 개발

### 1.2 배경
- 매일 아침 뉴스 확인에 시간이 많이 소요됨
- AI 관련 뉴스를 놓치지 않고 싶음
- 글로벌 뉴스도 한국어로 편하게 보고 싶음

### 1.3 핵심 가치
- **시간 절약**: 큐레이션된 뉴스로 빠른 정보 습득
- **AI 중심**: AI 뉴스 최우선 배치
- **언어 장벽 제거**: 글로벌 뉴스 자동 번역

## 3. 기술 스택

### 3.1 Frontend
```
Next.js 14+     → React 기반 풀스택 프레임워크
Tailwind CSS    → 유틸리티 CSS
TypeScript      → 타입 안전성
```

### 3.2 Backend
```
Next.js API Routes  → 서버리스 API
Vercel Cron         → 스케줄링 (무료 1일 1회)
```

### 3.3 외부 API
```
NewsAPI.org         → 글로벌 뉴스 (무료 100회/일)
네이버 뉴스 API     → 한국 뉴스 (무료 25,000회/일)
Google Translate    → 번역 (유료) 또는
LibreTranslate      → 번역 (무료 오픈소스)
```

### 3.4 배포
```
Vercel              → 무료 호스팅 + 자동 배포
```
